# GenAI Learning Lab

A hands-on repository documenting my journey from basic Retrieval-Augmented Generation (RAG) systems to production-style AI agents.

Rather than relying solely on tutorials, this repository focuses on implementing AI systems from scratch to understand how modern GenAI applications work internally.

## Learning Objectives

* Build Retrieval-Augmented Generation (RAG) systems
* Implement conversational memory
* Create tool-using agents
* Design planning and routing agents
* Build reflective and self-correcting systems
* Implement code execution agents
* Understand workflow orchestration and agent architectures
